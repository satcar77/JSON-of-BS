# nepali calender JSON


JSON datas of Nepali Calender(2065-2074). Weekends are not marked as holidays. Raise any issues to request any changes or modification to the file to fit your application demand. :+1:


Format :

```json
{"Baishakh": [
{"np": "31",
"en": "14", 
"tithi": "Tritiya", 
"event": "", 
"day": "Sun", 
"holiday": false},
....],
"Jestha":[...]
...
}
```
